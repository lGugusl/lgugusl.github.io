The document accessible at this address is a documentation/assessment I did right after the release of Wwise version 2019.2. While using Spatial Audio in my own project using Unity, and having fellow Audio team members using Wwise and Unreal and being unsue about which functions to use or how, I built this experiments and assessment alongside some explanations about the update (that complete a sharing about Spatial Audio technology I did within this team).
This document consist of a main text explaining and reporting the results for reference and future consultation, and files (found here) related to the experiments done in Unity and Unreal Engine, with raw notes text files, screenshots and Wwise profiling data files with all the recorded results. You can also find in there videos for comparisons about paths diffraction.
